# Static Mod Drop Files

These files follow the naming pattern used by dropped NCT governor mods:

- `2023KYGov_init.html`
- `2023KYGov_BeshearColeman.html`
- `2023KYGov_CameronMills.html`

To submit this like a standard dropped mod, add the object in `mods_json_entry.json` to the site's mod list and place these three HTML files in the static mod folder. The root `code1_2023_ky_governor.js` and `code2_2023_ky_governor.js` remain the custom-loader versions.
